# Ontograph

An ontology based DAG graph using cytoscape.js and the dagre layout, inspired by BioJS DAGViewer


## Installation

Build

    gulp

Build watch

    gulp watch

Build debug

    gulp devmode

Build watch debug

    gulp watchdev


Build ontologies

    bin/build.sh

Note: The debug mode is currently required for cose-bilkent
